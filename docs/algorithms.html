Neural Ridge Classifier¶

Also known as classical Extreme Learning Machine. Features from training instances are chosen randomly.

class pyridge.neural.NeuralRidge[source]¶

Neural Ridge classifier, also known as Extreme Learning Machine. It works as a single hidden layer neural network where neuron’s weights are chosen randomly.
