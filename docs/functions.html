J codification for targets¶

pyridge.utils.j_encode(targets, n_targ=None)[source]¶

Ridge classifier needs the targets are vectors: as many components as possible targets. The position of the vector which coincide with the number of the target is filled with 1, and the rest of the components is 0.

For example, if the target is binary, 1 and 2, the target for an instance is a vector [1, 0] if the target is 1, and [0, 1] is the target is 2.

Parameters:
  • targets – the array of targets.
  • n_targ – number of different targets. It reduce the number of operations.
Return T:

matrix of targets.
